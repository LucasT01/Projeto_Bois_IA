## \# Dataset — Rayvora Vision Pro

# 

# Dataset utilizado tanto no treinamento do Edge Impulse quanto no Fine-Tuning.

# 

# \## Conteúdo

# 

# Imagens de bovinos com respectivos labels de peso real (kg), usadas para o

# treinamento de regressão de peso via visão computacional (vista traseira).

# 

# \## Tamanho

# 

# Aproximadamente 2.5 GB (arquivo: Nosso\_data\_set.zip)

# 

# \## Por que não está no Git

# 

# O arquivo excede o limite de 100 MB do GitHub e não é viável usar Git LFS

# para este volume. O dataset fica armazenado no Google Drive.

# 

# \## Como obter

# 

# Acesse o link abaixo para baixar o dataset completo:

# 

# 🔗 \[Download do Dataset — Google Drive](https://drive.google.com/drive/folders/1zQGbpBTyy9ZNKG5x\_H\_qknaZ55PNSQX6?usp=sharing)

# 

# > Certifique-se de que tem permissão de acesso. Em caso de dúvidas,

# > entre em contato com a equipe (Ludivino José Da Silva).

# 

# \## Estrutura esperada após download

#
