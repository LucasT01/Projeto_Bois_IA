# Dataset - Rayvora Vision Pro 
 
Dataset usado tanto no treinamento do Edge Impulse quanto no Fine-Tuning. 
 
## Conteudo 
Imagens de bovinos com respectivos labels de peso real (kg), usadas para o treinamento 
de regressao de peso via visao computacional. 
 
## Tamanho 
Aproximadamente 2.5 GB (arquivo: Nosso_data_set.zip) 
 
## Por que nao esta no Git 
O arquivo excede o limite de 100 MB do GitHub e nao compensa o uso de Git LFS 
para este caso. O dataset fica armazenado localmente / em storage externo. 
 
## Como obter 
Contate o responsavel pelo projeto (Ludivino) para acesso ao arquivo Nosso_data_set.zip.
